== Frequently asked questions ==


= Is it possible to make multiple pricing table at a time? =
Yes you can make unlimited pricing plan by using same shortcode again.

= Can I link to pricing table button to an external link? =
Yes you can. It has a shortcode attribute for doing that.

= How many features I can add in pricing plan? =
It's unlimited. Just write comma separated features in shortcode attribute.
